# The Sky's Recipe / Glow-Box Mystery  
## Revised Discovery-Based Activity Flow

This revised flow prevents concept dumping at the end. Each science word is introduced only after the child has already discovered the idea through action.

---

## Stage 1: Make Any Glow

### Goal
Child tries freely and makes the bulb glow once.

### Screen
Show the following items on the workbench:

- Battery
- Bulb
- Wires
- Switch

The switch should be visible, but optional at first. For Grade 3-5, too many required parts at the start can create frustration. First, let the child make a simple glow using only the battery, bulb, and wire.

### Prompt
**Meera:** "Can you make this tiny bulb glow?"

### Child Action
Child connects the battery, wire, and bulb.

### Possible Outcomes

| Child action | Result |
|---|---|
| One side connected | Bulb stays dark |
| Wrong or incomplete connection | Bulb stays dark |
| Complete connection | Bulb glows |

### Nudges If Stuck

**Kabir:** "The bulb may need two touches, not one."  
**Meera:** "Can you give it a way to go in and come back?"  
**Kabir:** "Try touching both sides of the battery."

### Important Rule
Do not say **current** or **circuit** yet.

---

## Stage 2: Notice the Travelling Thing

### Trigger
When the bulb first glows, show small glowing dots moving from the battery, through the wire, into the bulb, and back.

### Dialogue
**Meera:** "Wait... something is travelling!"  
**Kabir:** "It goes around... and the bulb shows it as light."

### Question on Screen
**What should we call the travelling thing?**

### Playful Options

A. Glow traveller  
B. Light runner  
C. Battery sparkle

Let the child choose any option. There is no wrong answer.

### Confirmation
**Kabir:** "Yes! It is like a travelling glow thing."

### Introduce the Science Word
**Meera:** "Scientists call this travelling thing current."

### Tiny Concept Card
**Current = the travelling thing that moves through the wire.**

### Immediate Reuse
**Kabir:** "Can you make the current travel again?"

The child reconnects or taps to replay the glow.

### Why This Works
Current is named right after the child sees movement, not at the end.

---

## Stage 3: Kill the Glow

### Goal
The child discovers that the glow stops when the path is broken.

### Prompt
**Meera:** "Can you stop the glow without removing the bulb?"

### Child Action
Child pulls one wire away.

### Result
- Bulb turns off.
- Moving dots stop.
- A gap is highlighted.

### Dialogue
**Kabir:** "The current stopped!"  
**Meera:** "Why did it stop?"

### Question on Screen
**What happened to the path?**

### Options

A. The path broke  
B. The bulb got sleepy  
C. The battery disappeared

### Correct Choice
**The path broke.**

### Confirmation
**Kabir:** "Yes. The current needs a full path."

### Important Rule
Still do not name **circuit** yet. Let the child revive the glow first.

---

## Stage 4: Revive the Glow

### Prompt
**Meera:** "Can you fix the path and bring the glow back?"

### Child Action
Child reconnects the wire.

### Result
- Bulb glows again.
- Moving dots move again.

### Dialogue
**Kabir:** "You repaired the path!"  
**Meera:** "Now the current can travel all the way around."

### Introduce the Second Concept
**Kabir:** "A full, unbroken path has a science name."  
**Meera:** "It is called a circuit."

### Tiny Concept Card
**Circuit = a complete path for current.**

### Constellation Link
**Kabir:** "Like Orion only looks complete when we join the stars."  
**Meera:** "Our glow only works when the path is complete."

### Why This Works
This is the right place to name **circuit**, because the child has just broken and repaired the path themselves.

---

## Stage 5: Add the Switch as a Magic Path Door

### Goal
Introduce the switch as a control tool, not as a concept dumped at the end.

### Dialogue
**Kabir:** "Can we stop and start the glow without pulling the wire every time?"  
**Meera:** "Maybe this switch can help."

### Screen Change
The switch appears or becomes highlighted.

### Child Action
Child drags the switch into the circuit.

### Switch OFF Result
- Gap appears.
- Current dots stop.
- Bulb stays dark.

### Switch ON Result
- Path joins.
- Current dots move.
- Bulb glows.

### Dialogue
**Meera:** "The switch is like a tiny door in the path."  
**Kabir:** "Open door: current stops. Closed door: current travels."

### Concept Card
**Switch = opens and closes the circuit.**

### Why This Works
The child names the switch after using it as a control, instead of hearing about it in a final explanation.

---

## Stage 6: Build the Whole Glow Recipe

### Screen Reset
Reset the workbench.

This time, the child must build using:

- Battery
- Bulb
- Wires
- Switch

### Prompt
**Meera:** "Now make the full glow recipe."  
**Kabir:** "It needs a source, a path, and a way to start/stop."

Keep it playful. Avoid over-explaining.

### Child Action
Child builds the working circuit and switches it ON.

### Success Dialogue
**Kabir:** "Current is travelling!"  
**Meera:** "The circuit is complete!"  
**Kabir:** "And the bulb makes the current seeable as light!"

### Why This Works
This line links all three concepts only after the child has used each one.

---

## Stage 7: Closing Sky Callback

### Scene
Return to the night sky or classroom constellation wall.

Orion appears.

### Dialogue
**Meera:** "We first knew Orion's name."  
**Kabir:** "Now we know one glow recipe too."

They look at their tiny glowing bulb.

**Meera:** "Our bulb needed a source, a complete path, and current."  
**Kabir:** "A real star must have its own glow recipe."

Pause.

**Meera:** "But what powers a real star?"  
**Kabir:** "That sounds like our next mystery."

### End
End the activity here, with curiosity open for the next mission.

---

## Key Implementation Note

Do not explain **current**, **circuit**, and **switch** together at the end.

Use this order instead:

1. Child sees movement -> name **current**.
2. Child breaks and repairs the path -> name **circuit**.
3. Child uses switch to stop and start glow -> name **switch**.
4. Child rebuilds the full glow recipe -> link all three concepts together.
